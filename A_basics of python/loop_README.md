## Why Loops Matter in Scientific Computing

Loops are essential for:

- Iterating through observational datasets
- Applying conditional filters (e.g., selecting bright supernovae)
- Performing iterative numerical calculations
- Simulating physical processes step-by-step

Although NumPy vectorization is preferred for performance, 
understanding loops is fundamental for:

- Algorithm development
- Convergence testing
- Custom data processing routines
