## Why Dictionaries?

Dictionaries store data in key-value pairs, making them ideal for structured scientific information.

In astronomy and cosmology, dictionaries are useful for:

- Supernova properties (redshift, magnitude, host galaxy)
- Star parameters (mass, temperature, spectral type)
- Dataset organization
- Storing computed cosmological quantities
